백상갤러리 스토리보드 테스트

프로젝트 개요

: 이 프로젝트는 처음으로 도전한 클론 코딩이었습니다.

HTML, CSS, JavaScript를 활용하여 실제와 유사한 환경에서 웹 페이지를 구성하는 것은 매우 소중한 경험이라고 생각했습니다.

이를 위해 유명 웹사이트를 클론 코딩함으로써, 실제 사례를 통해 웹 디자인과 개발에서 마주할 수 있는 다양한 문제를 해결하는 능력을 배양할 수 있었습니다.

기술 스택

· HTML

· CSS

· JavaScript

· Jquery

· AOS

기능 구현

· 기능구현01

"왼쪽 상단 아이콘을 클릭하면, 매끄럽게 왼쪽에서 오른쪽으로 슬라이드되는 사이드바를 구현했습니다. 기존 심플했던 디자인을 액자 느낌의 디자인으로 구현했습니다."

· 기능구현02

"기존 사이트는 옆으로 자동 슬라이드되지 않고 정지해 있었습니다. 슬라이딩 효과를 추가하여 시각적인 즐거움을 높였습니다."

· 기능구현03

"아코디언 메뉴를 사용하여 사용자가 편리함을 높였습니다. 정보에 쉽게 접근이 가능합니다."

· 기능구현04

"아코디언 메뉴를 사용하여 사용자가 편리함을 높였습니다. 정보에 쉽게 접근이 가능합니다."

"마우스를 올릴 때 애니메이션 효과를 추가해 디자인에 생동감을 더했습니다."

· 기능구현05

"다른 이미지를 클릭하면 배경이 변경되도록 하여 사용자가 다양한 구도에서 콘텐츠를 감상할 수 있도록 했습니다."

더 다양한 기능은 제작 페이지에 방문하시면 확인하실 수 있습니다.

개선 및 배운점

스와이퍼 플러그인을 통해 여러 콘텐츠를 효율적으로 배치하는 방법을 익혔습니다.

CSS를 사용하면서 HTML의 튼튼한 구조가 얼마나 중요한지 깨달았습니다. 제대로 설계된 HTML은 CSS를 보다 쉽고, 유지보수와 확장성에서도 큰 도움이 되는걸 느꼈습니다.
